# Example Guides

Each folder in `examples/` is a ready-to-run micro-frontend scenario. Open the folder, install, and run.

- [basic/README.md](basic/README.md) — React shell + React counter remote, shared state and hooks
- [multi-framework/README.md](multi-framework/README.md) — React shell + React products + Vue cart sharing cart state
- dashboard/README.md — Coming soon: production-style multi-remote dashboard with shared UI package

## Prerequisites
- Node.js 18+ and pnpm installed

## Quick Start
```bash
# Basic example
cd examples/basic
pnpm install
pnpm dev

# Multi-framework example
cd ../multi-framework
pnpm install
pnpm dev
```
